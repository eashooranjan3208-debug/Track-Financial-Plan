File Naming

Files inside the ZIP must follow this convention:

YYYYMMDD_HHMMSS_PAN_plan.json YYYYMMDD_HHMMSS_PAN_report.html YYYYMMDD_HHMMSS_transactions.xlsx YYYYMMDD_HHMMSS_portfolio.xlsx YYYYMMDD_HHMMSS_asset_allocation.xlsx

Per-customer files are matched by PAN in the filename. System-wide files are matched by PAN inside the file rows. Unrecognized files are skipped and listed in the results.
