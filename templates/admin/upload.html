File Guide

All files are optional per upload. Upload only what you have.

plan.json per customer

Imports goals, cashflow projections, family, assets. Archives the previous plan automatically.

report.html per customer

Stored and shown as the customer's plan report.

transactions.xlsx system-wide

Contains all customers. PAN maps rows to accounts. Duplicate rows are automatically ignored.

portfolio.xlsx system-wide

Today's portfolio value snapshot. Re-uploading the same date is safe (ignored).

asset_allocation.xlsx system-wide

Live fund-level allocation snapshot.

File naming convention:
YYYYMMDD_HHMMSS_PAN_filetype.ext
System-wide files do not include PAN in the name.
